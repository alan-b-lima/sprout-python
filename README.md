Sprout é um pequeno chat desenvolvido para aprimorar as habilidades práticas de programação em redes de computadores utilizando os protocolos TCP e UDP.

Sprout-Python é também fruto de um trabalho da disciplina de Redes de Computadores. O trabalho cobre os principais conceitos de comunicação cliente-servidor, manipulação de threads e criação de um sistema de chat.
